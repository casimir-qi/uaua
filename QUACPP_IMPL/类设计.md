# 1 按键启动程序的配置：
usr_2/hmiframe.ini
## 添加如下配置
hmiframe.ini 没有有run节点，需要添加。如果有run节点，则在节点下增加外部程序。用序号依次添加。
### 启动外部程序
需要写入外部程序的全路径
```
[run]
;1=程序路径、脚本
1=/usr/drivedream/bin/comp_ui
2=/usr/drivedream/bin/程序全路径
```
### 启动脚本
脚本可以放在指定路径，或者是脚本的全路径
```
[run]
;
1=tool1start.sh
```

# 2 放置按键的配置：
在添加外部程序配置后可以组合
usr_2/hmi.cfg

找到[setup] 节点
配置到8号按键，在 0000000a=go#More 前一行添加配置
```
00000008=go#popuptool
80000001=run#1
;80000001=run#2 
8000000a=back
```
